# Geospatial Analysis Projects

This repository contains a collection of geospatial analysis and environmental modelling projects using **GIS tools and Python-based workflows**.

The projects focus on spatial data processing, terrain analysis, and environmental modelling, combining traditional GIS software with programmatic approaches.

---

## Overview

Geospatial analysis plays a critical role in understanding environmental patterns and spatial relationships. This repository demonstrates:

- Spatial data processing and analysis  
- Integration of GIS tools with Python  
- Terrain and environmental modelling  
- Reproducible geospatial workflows  

The work combines both **visual GIS-based analysis** and **script-based automation**.

---

## Projects

### 🗺️ GIS and Spatial Analysis (`gis-spatial-analysis`)
Projects developed using ArcGIS and Python.

- Spatial data visualization and mapping  
- Raster and vector data processing  
- Spatial queries and analysis  
- Integration of GIS tools with Python scripting  

👉 Focus: GIS workflows, spatial analysis, and mapping  

---

### 🌍 Environmental Modelling (`environmental-modelling`)
Projects using WhiteboxTools and Python for terrain and environmental analysis.

- Terrain analysis (e.g., elevation, slope, hydrology)  
- Environmental modelling workflows  
- Automated geospatial processing using Python  
- Reproducible analysis pipelines  

👉 Focus: environmental modelling, terrain analysis, and automation  

---
